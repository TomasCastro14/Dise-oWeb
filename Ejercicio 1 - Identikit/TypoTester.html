TypoTester

TypoTester es una herramienta interactiva diseñada para explorar y comparar diferentes tipografías dentro del entorno de Visual Studio Code. La página está dividida en varias secciones que permiten a los usuarios conocer y probar fuentes Serif, Sans-Serif, Monoespaciadas y Cursivas/Fantasy, visualizando de manera clara cómo se ven y se comportan en distintos contextos de diseño y programación. Además, cuenta con una guía para comprender mejor cómo elegir la fuente ideal para cada proyecto, ya sea para mejorar la legibilidad del código o simplemente personalizar la apariencia del entorno de trabajo.
